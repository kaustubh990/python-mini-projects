# Python Mini Projects

This repository contains beginner-friendly Python projects developed while learning Python programming fundamentals.

## Projects Included

### 1. Password Generator

A simple Python application that generates secure random passwords based on the length specified by the user.

#### Features
- User-defined password length
- Random combination of letters, numbers, and special characters
- Easy to use through the terminal

#### Concepts Used
- User Input
- Loops
- Random Module
- String Manipulation

---

### 2. Guess the Number

An interactive number-guessing game where the user tries to guess a randomly generated number within a certain range.

#### Features
- Random number generation
- User feedback for higher/lower guesses
- Tracks attempts until the correct number is guessed
- Beginner-friendly command-line game

#### Concepts Used
- Conditional Statements
- Loops
- Random Module
- User Input Handling

---

## Technologies Used

- Python 3
